# mlso-theme

[![MELPA](http://melpa.org/packages/mlso-theme-badge.svg)](http://melpa.org/#/mlso-theme)

A medium contrast theme for emacs with dark colors.

- Company
- Dired
- Eshell
- Flycheck
- Ido
- Magit
- Which-Key
